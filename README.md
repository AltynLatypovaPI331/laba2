# Лабораторная работы №2
## Выполненные задания:
1. Изучение теоретической части.
2. Создание репозитория на Github.
3. Выполнение всех примеров кода из лабораторной работы.
4. Реализация функции для вычисления возраста пользователя.
5. Реализация поиска кратных чисел представления.
6. Реализация функции, которая возвращает нечетные числа из диапазона.
7. Релизация игры «камень-ножницы-бумага».
8. Загрузка всех файлов в созданные каталоги репозитория.
## Дополнительная информация
Задания были реализаваны с помощью интегрированной среды разработки Spyder.
Были использованы стандартные функции Python для выполнения арифметических оперций.
В программах предусмотрены проверки для корректного ввода данных пользователем.
